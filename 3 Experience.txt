Experience Page Structure
The Experience page should display four cards, each representing one internship. Each card should show the organization name and my role. When a user clicks on a card, an expandable section should appear on the same page displaying detailed information about that internship.
The expandable section should include the following details: Role, Organization, Time Period, Mode of Work, Organization Context, Role Overview, Key Contributions, Skills Used, and Tools Used.
At the bottom of the expandable section, there should be a button labeled “See Work Samples.” When the user clicks this button, it should open a separate page dedicated to that internship, where I will provide Google Drive links to the work samples along with a short explanation of each piece of work.
The goal is to keep the Experience page clean and easy to scan, while allowing visitors to explore more detailed information and work samples if they are interested.
